 	Entrar Plataforma 
Cenário  Principal

1. Utilizador abre Programa
2. Programa mostra Página de Login
3. O Utilizador introduz os dados para realizar o login;
4. A app verifica se os dados estão corretos e válida a operação;
5. Plataforma mostra página "Home" (Com possibilidade de Gerir Atletas e Gerir Eventos)

Cenário secundário
1.1 O Utilizador introduz os dados incorretos
1.1.2 Aparece mensagem de erro "Dados introduzidos inválidos!"
1.2 O Utilizador carrega no botão "Sair";
1.2.1 Programa fecha

 	Criar Evento 
Cenário  Principal
(Pré-Condição: Utilizador encontra-se na página "Home")

1 Utilizador escolhe opção "Criar Evento"
2 Plataforma mostra-lhe janela para inserir dados do evento
3 Utilizador insere os dados pretendidos para o evento
4 Utilizador clica em "Criar Evento"
5 Plataforma regista evento e abre-o para os atletas poderem se inscrever

Cenário Secundário

1.1 Utilizador seleciona "Sair"
1.1.1 Programa fecha
3.1 Utilizador seleciona "Cancelar"
3.1.1 Plataforma volta à página "Home"
5.1 Dados inseridos inválidos
5.1.1 Plataforma dá aviso sobre os campos inseridos inválidos

 	Lançar Evento 
Cenário  Principal
(Pré-Condição: Utilizador encontra-se na página "Evento")

1 Utilizador Escolhe Opcao Lançar Evento
2 Plataforma abre janela de confirmação
3 Utilizador confirma lançamento do evento
4 Plataforma confirma estar condições todas válidas para lançar evento
5 Plataforma chama a Importacao de Atletas Inscritos, chama Criação das Pools
6 Evento é criado ficando aberto a inscrições

Cenário Secundário

1.1 Utilizador seleciona "voltar"
1.1.1 Plataforma volta à página "Home"
3.1 Utilizador seleciona "Cancelar"
3.1.1 Plataforma volta à página "Evento"
4.1 Por alguma incompatibilidade, evento não pode ser lançado (ex. falta de inscritos)
4.1.1 Plataforma informa Utilizador que evento não pode ser lançado e qual a razão dessa impossibilidade

 	Importar Evento(s) 
Cenário  Principal
(Pré-Condição: Utilizador encontra-se na página "Home")

1 Utilizador escolhe opção "Importar Eventos"
2 Plataforma mostra janela com opçao para fazer upload de ficheiros com os eventos a serem importados
3 Utilizador clica no botao "Procurar ficheiro"
4 Plataforma mostra janela para escolha do ficheiro
5 Utilizador escolhe ficheiro pretendido a importar
6 Utilizado clica em "Importar"
7 Plataforma valida ficheiro
8 Plataforma importa dados do ficheiro
9 Plataforma indica que a importação foi bem sucedida e evento aparecerá na lista


Cenário Secundário

1.1 Utilizador seleciona "Sair"
1.1.1 Plrograma fecha
3.1 Utilizador seleciona "Cancelar"
3.1.1 Plataforma volta à página "Home"
3.2 Utilizador cola ou escreve caminho para o ficheiro
3.2.1 Continua no ponto 6
7.1 Plataforma verifica que o ficheiro é inválido
7.1.1 Plataforma indica "ficheiro inválido" ao utilizador
